Project overview
This project is about visualizing a complex system â€“ the isolation of the elderly. The goal of this project is to visualize the problem in a comprehensive and presentable way and to cultivate systematic thinking.

Through incorporating the significance, external and internal causes, effects, and interventions of the problem, were able to reflect the most important aspects of this problem. More importantly, we learned how design is within the larger social, economic and political context, and how learning about other systems can help us generate better solutions.

02 Process



2.1 Initial Exploration

We used sticky notes to brainstorm and sort out different aspects of the problem. The color of the sticky notes represented the different categories. The in-between lines showed how each part is connected. This activity made us understand the problem as well as the logic behind it, which laid a foundation for the later visualization.

2.2 Iterations

Before we landed on the final version, we explored a lot of ways to present the problem so that it appeared logical and clear to the audience.
